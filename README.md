# 拼图游戏 (Jigsaw Puzzle Game)

一个基于 Java Swing 开发的滑块拼图游戏，支持登录注册、多图选择和步数统计。

## 功能特性

- **用户系统**: 登录/注册功能，新用户自动注册
- **拼图玩法**: 4x4 滑块拼图，通过方向键移动拼图
- **图片选择**: 提供三套图片主题（城市、火车、超级跑车）
- **步数统计**: 实时显示移动步数
- **胜利判定**: 通关后弹出胜利提示
- **快捷键**:
  - `↑ ↓ ← →`: 移动拼图块
  - `A`: 查看完整原图
  - `W`: 一键获胜（Debug 功能）
- **菜单功能**:
  - 重新开始
  - 重新登录
  - 退出游戏

## 运行方式

### 方式一： IDE 运行

在 IDE 中直接运行 `Intro.java` 入口类。

### 方式二： 命令行编译运行

```bash
cd src
javac Intro.java
java Intro
```

## 游戏规则

1. 游戏开始时，15 张图片块随机打乱（其中一块留空）
2. 使用键盘方向键移动图片块，使其按 1-15 顺序排列
3. 最后一块为空白格
4. 任意时刻按 `A` 键可查看完整原图作为参考
